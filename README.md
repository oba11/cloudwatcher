cloudwatcher
============

Submit memory and disk space usage metrics as Amazon CloudWatch Custom Metrics.

Available metrics include:

- Memory Utilization (%)
- Memory Used (MB)
- Memory Available (MB)
- Swap Utilization (%)
- Swap Used (MB)
- Disk Space Utilization (%)
- Disk Space Used (GB)
- Disk Space Available (GB)
